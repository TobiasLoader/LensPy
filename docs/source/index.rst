LensPy docs
==================================

Python library for the *Lens Protocol*. Opening up web3 to Python developers.

See the Lens `Protocol documentation <https://docs.lens.xyz/docs>`_.

.. image:: /_static/img/lenspylogo.png
   :alt: lens py logo

Aims for this project:
----------------------------------

- Make Lens Protocol easy to use and accessible to Python developers.
- Build demo projects with common frameworks like Flask, Django etc...
- Write good documentation for LensPy
